# 亚洲盘集中度判断工具 - Web应用

## 功能说明

输入A-R列的数据，判断是否满足以下高集中度条件：
1. **条件1**：除去走盘的，集中度≥85%（总场次≥6）
2. **条件2**：算上走盘的，集中度≥80%（总场次≥5）

仅支持判断 **主/0/0** 和 **客/0/0** 形态。

## 安装依赖

项目已配置虚拟环境，首次使用需要安装依赖：

```bash
# 创建虚拟环境（如果还没有）
python3 -m venv .venv

# 激活虚拟环境并安装依赖
source .venv/bin/activate
pip install flask
```

## 启动应用

### 方法1：使用启动脚本（推荐）
```bash
./start.sh
```

### 方法2：手动启动
```bash
source .venv/bin/activate
python3 app.py
```

然后在浏览器中访问：http://localhost:5000

## 使用说明

1. 在输入框中填写A-R列的数据
2. **必填字段**：B（主客）、D（澳门）、F（马会）
3. 点击"判断"按钮
4. 查看结果：绿色表示满足条件，红色表示不满足条件

## 列说明

- **A**: 强队
- **B**: 弱队/主客（必填：主 或 客）
- **C**: 时间
- **D**: 澳门（必填：0）
- **E**: 上水
- **F**: 马会（必填：0）
- **G**: 上水（红色列）
- **H**: 盘差
- **I**: 水差（红色列）
- **J**: 澳
- **K**: （红色列）
- **L**: 
- **M**: 马
- **N**: （红色列）
- **O**: 
- **P**: 主差（红色列）
- **Q**: 平差（红色列）
- **R**: 客差（红色列）

## 注意事项

- 当前仅支持主/0/0和客/0/0形态（D和F必须为"0"）
- 数值列（G, I, K, N, P, Q, R）可以留空，但会影响判断准确性
- 应用启动时会预加载所有规则，可能需要几秒钟
